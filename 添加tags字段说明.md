## 🎯 具体位置

在您的JSON中，找到这一行：

```json
"data": {
  "name": "赛博坐牢模拟器",
```

在这之后立即添加：

```json
"data": {
  "name": "赛博坐牢模拟器",
  "tags": [
    "模拟器",
    "看守所",
    "中国",
    "女性",
    "文字冒险",
    "状态系统",
    "事件系统",
    "NPC系统",
    "知识库",
    "DataTable",
    "记忆增强插件",
    "时间跳过",
    "智能简化"
  ],
```

## ✅ 验证

添加后，确保：

1. ✅ `tags` 是数组类型 `[]`
2. ✅ `tags` 不是 `null` 或 `undefined`
3. ✅ 所有标签都是字符串
4. ✅ JSON格式正确（注意逗号）

## 🔧 如果不想使用标签

如果您不想添加任何标签，可以使用空数组：

```json
"tags": []
```

**重要**：即使不使用标签，也必须包含 `tags` 字段（即使是空数组），否则会导致导入错误。
